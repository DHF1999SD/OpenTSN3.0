本目录中包含OpenTSN3.0集中式控制器的所有代码，使用方式参考doc目录下的使用手册
1、arp：          该文件夹存放arp代理进程代码，用于实现arp代理功能。
2、cnc_api：      该文件夹存放控制器的API库文件，提供数据接收、数据发送、TSMP协议等相关库函数，最后生成libcnc_api.a供其他应用程序调用。
3、cnc_ptp：      该文件夹存放时间同步应用代码，用于实现时间同步功能。
4、libxml2-2.6.2：该文件夹为libxml2库源文件，用户可以使用该源文件安装libxml库。
5、net_init：     该文件夹存放网络初始化应用代码，用于实现网络配置功能。
6、state_monitor：该文件下存放状态监测应用代码，用于实现状态检测功能
